# 📊 Customer Conversion Prediction

Group project for **LZSCC.460 – Data Science Fundamentals**.

## 🎯 Overview
This project analyzes a digital marketing dataset to identify key drivers of customer conversion and build predictive models for future campaigns.

## ❓ Research Question
1. What drives customer conversion?
2. Is predicting future conversions possible using the available customer data?

## 🧩 Objectives
1. **Preprocessing** – Clean data, handle missing values, detect biases  
2. **EDA** – Explore patterns and relationships using visualizations  
3. **Statistical Modelling** – Explain conversion drivers  
4. **Predictive Modelling** – Compare ML models (Elastic Net, Random Forest, XGBoost)  

## 📁 Structure
├── 01_preprocessing/

├── 02_eda/

├── 03_statistical_modelling/

├── 04_modelling/

├── utils/

├── marketing.csv

└── README.md


## ⚙️ How to Run
Run scripts in order:
1. `01_preprocessing`
2. `02_eda`
3. `03_statistical_modelling`
4. `04_modelling`

## 🧠 Methods
- Logistic Regression (Elastic Net)  
- Random Forest  
- XGBoost  
- Model evaluation (e.g., ROC)

## 🛠️ Requirements
R (≥ 4.0) with packages:
`dplyr`, `ggplot2`, `caret`, `pROC`, `PRROC`, `glmnet`, `randomForest`, `xgboost`, `broom`
